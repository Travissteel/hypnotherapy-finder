# Hypnotherapy Finder - Production Directory

A modern, SEO-optimized directory for finding certified hypnotherapists across the United States.

**Live Site**: https://hypnotherapy-finder.com

## Features

- 🔍 **Advanced Search & Filtering** - Search by location, specialty, and keywords
- 📍 **31 Location Pages** - Dedicated landing pages for major US cities
- 👤 **1,150+ Practitioner Profiles** - Comprehensive profiles with contact information
- 🎯 **SEO Optimized** - Schema.org markup, FAQ schema, sitemap, and optimized meta tags
- 📱 **Mobile Responsive** - Fully responsive design for all devices
- ♿ **WCAG 2.1 AA Compliant** - Accessible to all users
- ⚡ **High Performance** - Built with Next.js 16 for optimal speed
- 📧 **Email System** - Resend-powered transactional emails (claim notifications, welcome)
- 🏷️ **Backlink Badge System** - Verified practitioner badges for SEO backlinks
- 🔐 **Claim System** - Practitioners can claim and manage their listings

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Components**: shadcn/ui
- **Icons**: Lucide React
- **Database**: Supabase (PostgreSQL)
- **Auth**: Supabase Auth
- **Email**: Resend with React Email templates
- **Deployment**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deployment to Vercel

```bash
npx vercel --prod
```
