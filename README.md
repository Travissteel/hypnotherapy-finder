# Hypnotherapy Finder - Production Directory

A modern, SEO-optimized directory for finding certified hypnotherapists across the United States.

## Features

- 🔍 **Advanced Search & Filtering** - Search by location, specialty, and keywords
- 📍 **30+ Location Pages** - Dedicated landing pages for major US cities
- 👤 **2,000+ Practitioner Profiles** - Comprehensive profiles with contact information
- 🎯 **SEO Optimized** - Schema.org markup, sitemap, and optimized meta tags
- 📱 **Mobile Responsive** - Fully responsive design for all devices
- ♿ **WCAG 2.1 AA Compliant** - Accessible to all users
- ⚡ **High Performance** - Built with Next.js 14 for optimal speed

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Components**: shadcn/ui
- **Icons**: Lucide React
- **Deployment**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deployment to Vercel

```bash
npx vercel --prod
```
